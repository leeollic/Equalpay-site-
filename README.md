 Smart Hotel AI
This project is an AI-based hotel recommendation system.
It helps users choose the best room based on their budget and preferences.

Features
- Intelligent room recommendation system
- Budget-based filtering
- Preference-based scoring (cheap, luxury, balanced)
- Simple booking simulation

How it works
The system uses a custom scoring algorithm to evaluate available rooms and recommend the most suitable option based on user input.
